# @robinhood-trading/trading-service

Core trading logic and strategy engine for the Robinhood trading bot.

## Purpose

This package contains the business logic for:
- Order validation and risk management
- Position tracking and portfolio management
- Trading strategies (momentum, mean-reversion, etc.)
- Stop-loss and take-profit automation
- Trade history and analytics

## Agent Task

This package can be worked on independently by agents focusing on:
- Implementing risk management rules
- Creating trading strategies
- Adding portfolio analysis
- Building order validation logic
- Creating performance tracking
- Implementing stop-loss automation

## Development

```bash
npm run build
npm run test
npm run watch
```
